# Drum Game 

A Godot implementation of the game DTXMania.

## Introduction

This game aims to make the custom songs from the game DTXMania playable nativly in multiple platforms.

## Features
- Play dtx files created in the DTXCreator
- Auto load songs
- Keyboard, MIDI and controller input
- Playable in Windows, Linux and Mac
- Multiple profiles for each controller type

## Future Features
- Calibrator 
- Responsive UI
- App for editing song files
